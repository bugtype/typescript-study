# 1회차\(2021.07.27 화\)



1. 구조적 타이핑 언어\(**Structural Type System\)**이다.

```typescript
interface Btype {
  a:string;
  b:string;
}
interface Ctype {
  a:string;
  b:string;
}

const b : Btype = {a: 'hello', b: 'foo'};
const a : Ctype = b;

interface 이름이 다르더라도,property가 같으면 같은 type으로 인식한다.
(Structural Type System)
```
